  <h3>🚀 Flagship Project: Day-Night Crime Transmission Networks</h3>
  <p>
    An open-source research framework that analyzes 7 years of urban crime records through weighted directed graphs. 
    Built using <b>Python (NetworkX, Pandas, GeoPandas)</b> to identify temporal reconfigurations in crime patterns.
  </p>
  <ul>
    <li><b>Publication:</b> Peer-reviewed in Springer LNCS (MICAI 2025), nominated for Best Paper Award.</li>
    <li><b>Methodology:</b> Introduced an 8-state taxonomy and a temporal reconfiguration index (IR).</li>
    <li><b>Repository:</b> <a href="https://github.com/valeria-gm/Crime-Transmission-Networks">github.com/valeria-gm/Crime-Transmission-Networks</a></li>
